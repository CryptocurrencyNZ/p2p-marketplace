# p2p-marketplace

The Offical CryptoCurrencyNZ P2P marketplace!

## Installation

Install Dependencies

```
pnpm install
```

Setup dev secret key for AuthJS

```bash
# pnpm
pnpm exec auth secret

# npm
npm exec auth secret
```
